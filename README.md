## 🚀 **QUICK START**

### **Prerequisites**
- Node.js >= 20.0.0
- npm >= 10.0.0
- Python >= 3.10 (for some applications)

### **Start All Applications**
```bash
# Windows
./start-all-apps.bat

# Linux/Mac
./start-all-apps.sh
```

### **Individual Application Setup**
```bash
# Install dependencies
npm install

# Start specific application
cd applications/[app-name]
npm run dev
```
